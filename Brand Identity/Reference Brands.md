# Reference Brand Library

> 70 pre-extracted design systems from world-class brands. Drop any brand name into a request and Claude reads its `DESIGN.md` as context — exact colors, typography, spacing, surface logic, and visual atmosphere.
> **Parent:** [[Brand Identity/Brand Identity|Brand Identity]] · **Master:** [[CLAUDE]]

---

## How to Use

Tell Claude which brand's visual energy you want: *"Build this using the Starbucks DESIGN.md as reference"* or blend two: *"Starbucks warmth + Nike typographic authority."* Claude reads the file and applies that design DNA to the B&B brief.
